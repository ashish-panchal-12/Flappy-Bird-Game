# Flappy Bird Python

A simple Flappy Bird clone built with Python and Pygame.

## Features

* Classic Flappy Bird gameplay
* Animated bird sprites
* Random pipe generation
* Collision detection
* Sound effects
* Score tracking

## Project Structure

```
Flappy-bird-python/
│
├── assets/
│   ├── audio/
│   └── sprites/
│
├── flappy.py
├── Screenshot_1.png
└── README.md
```

## Requirements

* Python 3.x
* Pygame

## Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Flappy-bird-python.git
cd Flappy-bird-python
```

2. Install dependencies:

```bash
pip install pygame
```

## Run the Game

```bash
python flappy.py
```

## Controls

* **Spacebar**: Flap / Jump
